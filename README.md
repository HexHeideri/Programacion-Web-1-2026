# Fresh Aroma
Nuestra pagina se trata de una simple web de tienda la cual vende diferentes tipos de sahumerios.

Instrucciones:
Si se descargan los archivos, abriendo el "index.html" se puede empezar a navegar la pagina desde la "introduccion". En la parte de "Productos" se encuentra toda la presentacion de los productos, la tabla indicando aromas y demas de los diferentes items y un formulario para realizar el pedido. Luego en la seccion de "Contacto" esta la informacion necesaria para poder comunicarse ya sea a travez de redes sociales (links), de forma directa con un email o whatsapp, y por ultimo un formulario.

Tambien se puede acceder a la pagina a travez del link generado gracias a GitHub: https://hexheideri.github.io/Programacion-Web-1-2026/
